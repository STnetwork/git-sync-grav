```
• Notation system to avoid confusing number systems
• 1101 can be a Decimal, Hexadecimal or Binary value.
• Following applies when viewing numbers on computer :
	• 1101 by itself usually indicates a Decimal Value.
	• 0x1101 indicates a Hexadecimal value
	• 0b1100 indicates a Binary value.
```
